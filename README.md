# Thaman Dechamma A K - Portfolio Website

A professional portfolio website showcasing the work and experience of a Product Engineer specializing in secure file transfer systems, Linux server environments, web applications, and Android development.

## 📋 Overview

This is a modern, responsive portfolio website built with HTML5, CSS3, and vanilla JavaScript. It features a clean design with smooth animations and an intuitive navigation system.

## ✨ Features

- **Responsive Design**: Fully responsive layout that works on all device sizes
- **Smooth Navigation**: Sticky navbar with smooth scrolling between sections
- **Mobile Menu**: Toggle navigation for mobile devices
- **Fade-in Animations**: Intersection Observer-based animations for elements as they come into view
- **Contact Form**: Email integration using mailto links
- **Modern Styling**: Custom CSS with a warm color palette and gradient backgrounds

## 📁 Project Structure

```
website/
├── index.html          # Main portfolio page
├── style.css           # All styling
├── script.js           # Interactivity and animations
├── home.html           # (Currently unused)
└── README.md          # This file
```

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Custom CSS (no frameworks)
- **Interactivity**: Vanilla JavaScript with Intersection Observer API
- **Design Pattern**: Semantic HTML, progressive enhancement

## 📖 Sections

1. **Navigation Bar** - Sticky navbar with logo and navigation links
2. **Hero Section** - Introduction with CTA buttons and contact information
3. **About** - Career objective and skill categories (Frontend, Backend, Linux)
4. **Experience** - Professional experience as Product Engineer at NimbusParc TechnoLab

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No build tools or dependencies required

### Usage

1. Open `index.html` in your web browser
2. Navigate through sections using the navbar
3. Click on navigation links to smoothly scroll to sections
4. Use the contact form to send email inquiries
